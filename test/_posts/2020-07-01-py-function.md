# 함수(Function)
함수는 하나의 기능성을 가진 코드의 집합을 말합니다. 그리고 함수를 사용하는 것을 보통 **함수를 호출한다**라고 표현합니다. 함수를 호출할 때 괄호( ) 내부에 **매개변수(parameter)**라는 자료를 넣어 최종적으로 **리턴값**이라는 결과를 부릅니다.

## 함수의 기본 형태
```
def 함수 이름(매개변수1, 매개변수2, ...):
    문장
```

```python
def function(value, n):
    for i in range(n):
    print(value)
```
