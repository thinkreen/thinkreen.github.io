# 자료형(data type)
자료형이란 파이썬에서 기능과 역할에 따라 구분한 자료의 형식을 말합니다. 데이터의 자료형을 확인할 때는 `type()` 함수를 사용한다.
```python
>>> print(type("안녕하세요"))
<class 'str'>       # 문자열
>>> print(type(273))
<class 'int'>       # 정수
```
